# CHARLI Health

A warm, accessible VS Code colour theme featuring Taffy pink, Blueberry blue, and Jungle green, inspired by the CHARLI Health visual identity.

## Brand Colours

| Colour | Hex | Role |
|--------|-----|------|
| **Taffy** | `#F1A8C9` | Primary brand — keywords, accents (dark) |
| **Blueberry** | `#1D76B8` | Primary accent — buttons, badges, accents (light) |
| **Jungle** | `#34B9A4` | Success states, debugging |
| **Apricot** | `#FCC88F` | Constants, warnings, warm highlights |
| **Salmon** | `#F58F85` | Errors, tags, attention elements |
| **Almond** | `#FDE7D2` | Light theme panels, subtle backgrounds |

## Dark Theme

- **Background**: Deep plum (`#1E1A24`) for reduced eye strain
- **Text**: Warm off-white (`#F0E8E0`)
- **Keywords**: Taffy (`#F1A8C9`)
- **Functions**: Blueberry-light (`#4A9AD4`)
- **Strings**: Jungle-light (`#4DCDB8`)
- **Constants**: Apricot (`#FCC88F`)
- **Errors**: Salmon (`#F58F85`)

## Light Theme

- **Background**: Warm cream (`#FFF6ED`)
- **Text**: Charcoal (`#58595B`)
- **Keywords**: Blueberry (`#1D76B8`)
- **Functions**: Deep rose (`#9E4470`)
- **Strings**: Deep Jungle (`#1B7264`)
- **Constants**: Deep Salmon (`#B04840`)
- **Panels**: Almond (`#FDE7D2`)

## Installation

1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X / Cmd+Shift+X)
3. Search for "CHARLI Health"
4. Click Install
5. Select the theme: File > Preferences > Colour Theme
