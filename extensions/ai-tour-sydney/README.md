# AI Tour Sydney Theme

A modern, premium dark theme for Visual Studio Code inspired by the AI Tour Sydney 20-24 April 2026 event.

## Features

- **Deep navy backgrounds** (#0F2E74) for reduced eye strain
- **Vibrant cyan accents** (#C8F9F9) for excellent visibility
- **Brilliant blue highlights** (#1570EF) for code structure
- **Professional colour palette** derived from event branding
- **Optimised contrast ratios** meeting WCAG AA standards
- **Dark and light variants** for different coding environments
- **Full semantic token support** for modern editors

## Installation

1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X)
3. Search for "AI Tour Sydney"
4. Click Install

## Activation

1. Press `Ctrl+K Ctrl+T` to open the theme selector
2. Select "AI Tour Sydney Dark" or "AI Tour Sydney Light"

## Colour Palette

### Dark Theme
- **Background**: `#0F2E74` (Deep Navy)
- **Accent**: `#1570EF` (Brilliant Blue)
- **Keyword**: `#C8F9F9` (Cyan)
- **String**: `#B1DDF5` (Light Blue)
- **Comment**: `#7B95C8` (Muted Blue)

### Light Theme
- **Background**: `#F5F7FA` (Off White)
- **Accent**: `#0F5FD8` (Darker Blue)
- **Keyword**: `#0F5FD8` (Deep Blue)
- **String**: `#1E6784` (Teal)
- **Comment**: `#5B7B9C` (Muted Teal)
