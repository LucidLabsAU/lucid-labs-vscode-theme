# Aurora Dairies

A calm, modern VS Code colour theme inspired by the Aurora Dairies visual identity — pasture-based dairy farms across Australia and New Zealand.

## Brand Colours

| Colour | Hex | Role |
|--------|-----|------|
| **Aurora Blue** | `#003599` | Primary brand — status bar, buttons, badges, accents (light) |
| **Sky Blue** | `#5B8FE5` | Keywords, info, syntax (dark) |
| **Azure** | `#6FA0FF` | Functions, interactive accents (dark) |
| **Pasture Green** | `#7FC890` | Strings, success states (dark) |
| **Forest Green** | `#2D7A3F` | Strings, success states (light) |
| **Dawn Gold** | `#E0B870` | Constants, warnings (dark) |
| **Burnt Gold** | `#8B6B20` | Constants, warnings (light) |
| **Coral** | `#E07F58` | Tags, alerts |
| **Cream** | `#E6ECF5` | Foreground text (dark) |

## Dark Theme

- **Background**: Deep night blue (`#0C1A33`) — evoking the aurora sky
- **Text**: Soft cream (`#E6ECF5`)
- **Keywords**: Sky blue (`#5B8FE5`)
- **Functions**: Azure (`#6FA0FF`)
- **Strings**: Pasture green (`#7FC890`)
- **Constants**: Dawn gold (`#E0B870`)
- **Status bar**: Aurora Blue (`#003599`)

## Light Theme

- **Background**: Crisp white (`#FFFFFF`)
- **Text**: Deep night blue (`#0A1530`)
- **Keywords**: Aurora Blue (`#003599`)
- **Functions**: Aurora Blue (`#003599`)
- **Strings**: Forest green (`#2D7A3F`)
- **Constants**: Burnt gold (`#8B6B20`)
- **Status bar**: Aurora Blue (`#003599`)

## File Icon Theme

Includes a custom file icon theme featuring the Aurora Dairies "U with splash" mark for Git files.

## Installation

1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X / Cmd+Shift+X)
3. Search for "Aurora Dairies"
4. Click Install
5. Select the theme: File > Preferences > Colour Theme
