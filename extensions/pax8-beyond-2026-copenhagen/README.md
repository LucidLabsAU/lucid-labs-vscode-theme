# Pax8 Beyond 2026 Copenhagen

A bold, **red-forward** VS Code colour theme for **Pax8 Beyond 2026 Copenhagen** — the EMEA edition at the Bella Center, 18–20 October 2026. Same official Beyond palette as the Salt Lake City edition, with Community Red leading instead of Innovation Blue.

## Brand Colours

Sourced from the [Pax8 Beyond 2026 press kit](https://www.pax8beyond.com/emea/pax8beyond-presskit/).

| Colour | Hex | RGB | CMYK | Pantone | Role |
|--------|-----|-----|------|---------|------|
| **Community Red** | `#FF0D3B` | 255 / 13 / 59 | 0 / 98 / 75 / 0 | PMS 199C, 2035U | Primary accent — buttons, status bar, links |
| **Education Yellow** | `#ECFF18` | 236 / 255 / 24 | 13 / 0 / 100 / 0 | PMS 389C, 388U | Strings, attributes, highlights |
| **Innovation Blue** | `#00BBEF` | 0 / 187 / 239 | 68 / 4 / 0 / 0 | PMS 2985C, 297U | Keywords, info, tags |
| **Black** | `#000000` | 0 / 0 / 0 | — | — | Activity bar, deepest surfaces, banner |
| **White** | `#FFFFFF` | 255 / 255 / 255 | — | — | Editor background (light), text on accents |

> The brand colours focus on **red, yellow, blue, black and white**.

## Editions

- **Pax8 Beyond 2026** — Salt Lake City (NA), blue-forward → `lucidlabs.pax8-beyond-2026-theme`
- **Pax8 Beyond 2026 Copenhagen** — EMEA, red-forward → this extension

## Features

- Dual dark and light variants
- WCAG AA contrast for all syntax tokens (darkened red/yellow/blue used in the light variant for legibility)
- Brand-aligned activity-bar icon (the Pax8 cloud, in Community Red)
- Custom file icon theme using the brand palette
- Brand Palette sidebar with one-click HEX / RGB / CMYK copy

## Install

```bash
code --install-extension lucidlabs.pax8-beyond-2026-copenhagen-theme
```

Then pick **Pax8 Beyond 2026 Copenhagen Dark** or **… Light** from `File > Preferences > Theme > Color Theme`.
