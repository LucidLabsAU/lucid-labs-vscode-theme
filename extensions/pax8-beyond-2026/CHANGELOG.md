# Changelog

## [1.0.0] - 2026-06-06

### Added

- Initial release.
- Pax8 Beyond 2026 Dark and Light theme variants.
- Conference palette: Community Red (`#FF0D3B`), Education Yellow (`#ECFF18`)
  and Innovation Blue (`#00BBEF`) on black.
- WCAG AA contrast across syntax tokens and UI surfaces.
- Custom file icon theme using the Pax8 Beyond palette.
- Activity-bar brand mark (the Pax8 cloud) and Brand Palette sidebar with
  one-click HEX / RGB / CMYK copy and Dark/Light theme-role toggle.
