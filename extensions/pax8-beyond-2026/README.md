# Pax8 Beyond 2026

A high-energy VS Code colour theme for the **Pax8 Beyond 2026** conference — the official conference palette of Community Red, Education Yellow and Innovation Blue on black.

## Brand Colours

Sourced from the [Pax8 Beyond 2026 press kit](https://www.pax8beyond.com/pax8beyond-presskit/).

| Colour | Hex | RGB | CMYK | Pantone | Role |
|--------|-----|-----|------|---------|------|
| **Community Red** | `#FF0D3B` | 255 / 13 / 59 | 0 / 98 / 75 / 0 | PMS 199C, 2035U | Keywords, errors, badges |
| **Education Yellow** | `#ECFF18` | 236 / 255 / 24 | 13 / 0 / 100 / 0 | PMS 389C, 388U | Strings, attributes, highlights |
| **Innovation Blue** | `#00BBEF` | 0 / 187 / 239 | 68 / 4 / 0 / 0 | PMS 2985C, 297U | Primary accent — buttons, links, status bar |
| **Black** | `#000000` | 0 / 0 / 0 | — | — | Activity bar, deepest surfaces, banner |
| **White** | `#FFFFFF` | 255 / 255 / 255 | — | — | Editor background (light), text on accents |

> The brand colours focus on **red, yellow, blue, black and white**.

## Features

- Dual dark and light variants
- WCAG AA contrast for all syntax tokens (darkened red/yellow/blue used in the light variant for legibility)
- Brand-aligned activity-bar icon (the Pax8 cloud)
- Custom file icon theme using the brand palette
- Brand Palette sidebar with one-click HEX / RGB / CMYK copy

## Install

```bash
code --install-extension lucidlabs.pax8-beyond-2026-theme
```

Then pick **Pax8 Beyond 2026 Dark** or **Pax8 Beyond 2026 Light** from `File > Preferences > Theme > Color Theme`.
