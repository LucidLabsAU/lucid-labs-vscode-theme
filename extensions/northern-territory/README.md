# Northern Territory Theme

A professional VS Code colour theme featuring the Northern Territory's
iconic red ochre and black colour palette.

## Features

- **Red Ochre & Black Palette** — Northern Territory's signature colours throughout the editor
- **Dual Theme Support** — Both professional dark and clean light theme variants
- **System Theme Support** — Automatically follows your system's light/dark preference
- **WCAG AA Contrast** — All text colours meet or exceed accessibility standards
- **Custom Icon Theme** — Branded file icons for popular languages and tools

## Colour Palette

| Colour | Hex Code | Usage |
|--------|----------|-------|
| **Red Ochre** | `#C75B12` | Primary accent, functions, links |
| **Black** | `#000000` | Dark theme base, light theme keywords |
| **Amber** | `#FFB347` | Dark theme keywords, highlights |
| **Green** | `#6BBE27` | Strings, success states |
| **Light Blue** | `#5B9BD5` | Info, secondary accent |

## Installation

1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X / Cmd+Shift+X)
3. Search for "Northern Territory Theme"
4. Click Install
5. Go to File > Preferences > Colour Theme
6. Choose **Northern Territory Dark** or **Northern Territory Light**
