# Dark and Light — switch instantly

Lucid Labs ships two variants from the same brand palette.

| Variant | When to use |
|---------|-------------|
| **Dark** | Default coding environment, focus blocks, evening work |
| **Light** | Pair programming, screen sharing, client demos, bright rooms |

## Shortcuts

- `Cmd+Alt+L` / `Ctrl+Alt+L` — toggle the active variant
- `Cmd+K Cmd+T` / `Ctrl+K Ctrl+T` — open the full theme picker

## Auto-follow system theme

If you set VS Code's colour theme to **"Auto"** and pick Lucid Labs as both the dark and light theme, VS Code will follow your OS appearance automatically.
