# Lucid Labs file icons

The extension ships a matching icon theme so file explorers, breadcrumbs, and tabs all carry the brand.

## Coverage

- **Microsoft stack** — PowerShell, Bicep, ARM templates, C#, .NET
- **Web** — TS, JS, React, Vue, Svelte, HTML, CSS
- **Data** — JSON, YAML, CSV, Parquet, TMDL, SQL
- **DevOps** — Docker, Terraform, Makefile, GitHub Actions, Azure DevOps
- **Documents** — Markdown, PDF, Word, Excel, PowerPoint

Use the button to open the icon theme picker, then choose **Lucid Labs Icons**.
