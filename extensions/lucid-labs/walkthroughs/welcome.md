# Welcome to Lucid Labs

The Lucid Labs theme brings the official brand palette into VS Code — a deep purple base, teal accents, and warm off-white text designed to reduce eye strain over long coding sessions.

## What you get

- **Dark** and **Light** variants that both meet WCAG AA contrast
- **906 themed UI tokens** — every VS Code surface (editor, terminal, chat, notebook, debug, diff, status bar) carries the brand
- A matching **icon theme** for files and folders
- Soft, brand-aligned ANSI terminal palette tuned for `git`, `az`, `pwsh` output

Use the buttons in the description above to activate the variant that suits the room you're in.
