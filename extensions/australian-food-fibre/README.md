# Australian Food & Fibre

A professional VS Code colour theme inspired by the Australian Food & Fibre (AFF) visual identity. Features deep navy backgrounds with gold accents, evoking Australian cotton fields and agricultural excellence.

## Brand Colours

| Colour | Hex | Role |
|--------|-----|------|
| **Navy Blue** | `#1E3D82` | Primary brand — buttons, badges, keywords |
| **Gold** | `#C4A05C` | Accent — interactive elements, brackets (dark) |
| **Sky Blue** | `#5B8CE8` | Secondary — keywords, info (dark) |
| **Cotton Green** | `#8FBF7A` | Strings (dark), agricultural accent |
| **Warm Gold** | `#D4B06C` | Constants, warnings |
| **Terracotta** | `#D87048` | Tags, HTML elements |

## Dark Theme

- **Background**: Deep navy (`#0E1826`)
- **Text**: Cool off-white (`#E4E8F0`)
- **Keywords**: Bright blue (`#5B8CE8`)
- **Functions**: Light blue (`#6B9BFF`)
- **Strings**: Cotton green (`#8FBF7A`)
- **Constants**: Warm gold (`#D4B06C`)
- **Accent**: Gold (`#C4A05C`)

## Light Theme

- **Background**: Clean white (`#FFFFFF`)
- **Text**: Dark navy (`#111827`)
- **Keywords**: Navy blue (`#1E3D82`)
- **Functions**: Navy blue (`#1E3D82`)
- **Strings**: Forest green (`#3A6B2E`)
- **Constants**: Dark gold (`#8B6B20`)
- **Status Bar**: Navy blue (`#1E3D82`)

## Installation

1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X / Cmd+Shift+X)
3. Search for "Australian Food & Fibre"
4. Click Install
5. Select the theme: File > Preferences > Colour Theme
