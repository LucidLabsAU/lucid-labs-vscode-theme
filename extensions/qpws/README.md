# Queensland Parks & Wildlife (Unofficial)

An **unofficial**, community-built VS Code colour theme inspired by the earthy
palette of Queensland's parks and wildlife — deep teal, cream, and bark brown.

> **Not affiliated with or endorsed by the State of Queensland.** This is an
> independent theme published by Lucid Labs. "Queensland Parks and Wildlife
> Service" and the QPWS crest are trademarks of the State of Queensland and are
> **not** reproduced in this extension.

## Brand Colours

| Colour | Hex | Role |
|--------|-----|------|
| **Park Teal** | `#014F54` | Primary brand — status bar, buttons, badges, accents (light) |
| **Bright Teal** | `#3DB6AD` | Interactive accent, syntax, functions (dark, AA-contrast) |
| **Cream** | `#DED2BF` | Strings, terminal white, foreground tint |
| **Bark Brown** | `#784F23` | Keywords, tags (light), secondary warm tone |
| **White** | `#FFFFFF` | Editor background (light), text on teal |

## Features

- Dual dark and light variants
- WCAG AA contrast for all syntax tokens
- Queensland state-map silhouette activity-bar mark (Lucid Labs artwork)
- Custom file icon theme using the brand palette
- Brand Palette sidebar with one-click HEX / RGB / CMYK copy

## Install

```bash
code --install-extension lucidlabs.qpws-theme
```

Then pick **Queensland Parks & Wildlife Dark** or **… Light** from
`File > Preferences > Theme > Color Theme`.
