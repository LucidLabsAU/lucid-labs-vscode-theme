# Asplundh

A bold, high-contrast VS Code colour theme inspired by the Asplundh visual identity — the unmistakable orange-and-black livery seen on Asplundh utility vegetation management crews and trucks.

## Brand Colours

| Colour | Hex | Role |
|--------|-----|------|
| **Asplundh Orange** | `#F08028` | Primary brand — status bar, buttons, badges, accents (dark) |
| **Burnt Orange** | `#C75F10` | Primary brand — accents, syntax (light, AA-contrast) |
| **Hot Orange** | `#FF9540` | Hover state on accents |
| **Amber** | `#FFB870` | Constants, functions, attributes (dark) |
| **Black** | `#000000` | Activity bar, deepest surfaces (dark) |
| **Near-Black** | `#0F0F0F` | Editor background (dark) |
| **White** | `#FFFFFF` | Editor background (light), text on orange |

## Features

- Dual dark and light variants
- WCAG AA contrast for all syntax tokens
- Brand-aligned activity-bar icon (the Asplundh "A")
- Custom file icon theme using the brand palette
- Brand Palette sidebar with one-click HEX / RGB / CMYK copy

## Install

```bash
code --install-extension lucidlabs.asplundh-theme
```

Then pick **Asplundh Dark** or **Asplundh Light** from `File > Preferences > Theme > Color Theme`.
