# Victoria Theme

A professional VS Code colour theme featuring Victoria's
iconic blue and navy colour palette.

## Features

- **Blue & Navy Palette** — Victoria's signature colours throughout the editor
- **Dual Theme Support** — Both professional dark and clean light theme variants
- **System Theme Support** — Automatically follows your system's light/dark preference
- **WCAG AA Contrast** — All text colours meet or exceed accessibility standards
- **Custom Icon Theme** — Branded file icons for popular languages and tools

## Colour Palette

| Colour | Hex Code | Usage |
|--------|----------|-------|
| **Vic Blue** | `#0052C2` | Primary accent, functions, links |
| **Navy** | `#003174` | Keywords, types, attributes |
| **Green** | `#6BBE27` | Strings, success states |
| **Yellow** | `#FFCC2C` | Constants, warnings |
| **Light Blue** | `#5B9BD5` | Info, secondary accent |

## Installation

1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X / Cmd+Shift+X)
3. Search for "Victoria Theme"
4. Click Install
5. Go to File > Preferences > Colour Theme
6. Choose **Victoria Dark** or **Victoria Light**
