# Pax8 CTF 2026

A neon cyberpunk VS Code colour theme for the **Pax8 + Microsoft Beyond 2026 Capture the Flag** — terminal green and cyan on hacker dark.

## Brand Colours

Sampled from the live [pax8ctf.com](https://pax8ctf.com) design system (CSS custom properties).

| Colour | Hex | Role |
|--------|-----|------|
| **Terminal Green** | `#39FF14` | Primary accent — prompts, buttons, status bar, success |
| **Cyber Cyan** | `#29C5F6` | Keywords, info, tags |
| **Pax8 Blue** | `#1657FF` | Functions, links (brightened to `#5C8CFF` on dark) |
| **Signal Orange** | `#F25022` | Errors (Microsoft red) |
| **Caution Yellow** | `#FFB900` | Warnings, constants |
| **Hacker Dark** | `#0A0F1A` | Editor background |
| **Void** | `#05070D` | Activity bar, deepest surfaces, banner |
| **Terminal Text** | `#E6EDF3` | Primary text |

## Features

- Dual dark and light variants (dark is the headline — built for late-night flag hunting)
- WCAG AA contrast for all syntax tokens (darkened neon used in the light variant for legibility)
- Brand-aligned activity-bar icon (the Pax8 cloud)
- Custom file icon theme using the brand palette
- Brand Palette sidebar with one-click HEX / RGB / CMYK copy

## Install

```bash
code --install-extension lucidlabs.pax8-ctf-2026-theme
```

Then pick **Pax8 CTF 2026 Dark** or **Pax8 CTF 2026 Light** from `File > Preferences > Theme > Color Theme`.
