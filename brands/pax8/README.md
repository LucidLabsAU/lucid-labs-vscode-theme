# Pax8

A sharp, black-first VS Code colour theme inspired by the Pax8 visual identity — electric **Mint** green and deep **Pax8 Blue** on black.

## Brand Colours

| Colour | Hex | Role |
|--------|-----|------|
| **Mint** | `#03DE91` | Primary brand — accents, buttons, status bar, syntax (dark) |
| **Pax8 Blue** | `#0447BF` | Primary brand — accents, syntax (light, AA-contrast) |
| **Bright Mint** | `#2AE8A6` | Hover state on accents (dark) |
| **Sky Blue** | `#5AA0FF` | Keywords, links (dark) |
| **Black** | `#000000` | Activity bar, deepest surfaces, marketplace banner |
| **Near-Black** | `#0E0E12` | Editor background (dark) |
| **White** | `#FFFFFF` | Editor background (light), text on mint |

## Features

- Dual dark and light variants
- WCAG AA contrast for all syntax tokens
- Brand-aligned activity-bar icon (the Pax8 cloud)
- Custom file icon theme using the brand palette
- Brand Palette sidebar with one-click HEX / RGB / CMYK copy

## Install

```bash
code --install-extension lucidlabs.pax8-theme
```

Then pick **Pax8 Dark** or **Pax8 Light** from `File > Preferences > Theme > Color Theme`.
