# CHARLI Health Theme

A warm, accessible VS Code theme based on the CHARLI Health brand guidelines. Features Taffy pink, Blueberry blue, and Jungle green with both dark and light variants optimised for healthcare development.

## Dark Theme

- **Background**: Deep plum (#1E1A24) for reduced eye strain
- **Accent**: Taffy pink (#F1A8C9) for highlights and active elements
- **Functions**: Blueberry blue (#4A9AD4) for functions and operators
- **Strings**: Jungle teal (#4DCDB8) for strings and symbols
- **Constants**: Peach (#FCC88F) for numbers and constants

## Light Theme

- **Background**: Warm cream (#FFF6ED) for comfortable reading
- **Accent**: Blueberry blue (#1D76B8) for highlights and active elements
- **Functions**: Magenta (#9E4470) for functions and operators
- **Strings**: Deep green (#1B7264) for strings and symbols
- **Constants**: Burnt coral (#B04840) for numbers and constants

## Installation

1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X / Cmd+Shift+X)
3. Search for "CHARLI Health Theme"
4. Click Install
5. Select the theme: File > Preferences > Colour Theme
