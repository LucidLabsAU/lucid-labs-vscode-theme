# Progenesis

A professional, tech-forward VS Code colour theme featuring Progenesis blue, gradient teal, and magenta accents, inspired by the Progenesis visual identity.

## Brand Colours

| Colour | Hex | Role |
|--------|-----|------|
| **Progenesis Blue** | `#1F7AFF` | Primary brand — accents, buttons, functions |
| **Dark Blue** | `#062E73` | Keywords (light theme), deep accents |
| **Gradient Teal** | `#20DCD3` | Keywords (dark theme), brackets, highlights |
| **Gradient Pink** | `#DA79D0` | Constants, magenta highlights |
| **Warm Peach** | `#E8A87C` | Strings, HTML attributes |
| **Navy** | `#181C26` | Dark theme background |

## Dark Theme

- **Background**: Deep navy (`#181C26`) for a professional, focused coding environment
- **Text**: Cool off-white (`#E2E6EF`)
- **Keywords**: Gradient teal (`#20DCD3`)
- **Functions**: Light blue (`#5A9FFF`)
- **Strings**: Warm peach (`#E8A87C`)
- **Constants**: Soft magenta (`#DA79D0`)
- **Errors**: Coral red (`#F06060`)

## Light Theme

- **Background**: Cool white (`#F8F9FB`)
- **Text**: Dark charcoal (`#2A2D35`)
- **Keywords**: Dark blue (`#062E73`)
- **Functions**: Progenesis blue (`#1F7AFF`)
- **Strings**: Deep teal (`#1B7264`)
- **Constants**: Deep magenta (`#8E2580`)
- **Panels**: Light blue-grey (`#EDF0F5`)

## Installation

1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X / Cmd+Shift+X)
3. Search for "Progenesis"
4. Click Install
5. Select the theme: File > Preferences > Colour Theme
