# Perfection Fresh

A vibrant VS Code colour theme featuring red seal accents and fresh produce greens, inspired by the Perfection Fresh Australia visual identity.

## Brand Colours

| Colour | Hex | Role |
|--------|-----|------|
| **Red Seal** | `#CC2936` | Primary brand — accent, top bar, buttons |
| **Rose** | `#E8525E` | Keywords, tags (dark) |
| **Forest Green** | `#1B5633` | Strings, functions (light), borders |
| **Leaf Green** | `#6DBF7B` | Strings, success states (dark) |
| **Harvest Gold** | `#E8C870` | Constants, warnings |
| **Sky Blue** | `#5AABE0` | Info, terminal blue (dark) |

## Dark Theme

- **Background**: True black (`#141414`) — clean, modern dark mode
- **Text**: Warm off-white (`#E8E6E2`)
- **Keywords**: Rose (`#E8525E`)
- **Functions**: Red Seal (`#CC2936`)
- **Strings**: Leaf green (`#6DBF7B`)
- **Constants**: Harvest gold (`#E8C870`)
- **Errors**: Red (`#E05050`)

## Light Theme

- **Background**: Clean white (`#FFFFFF`)
- **Text**: Near-black (`#1A1A1A`)
- **Keywords**: Red Seal (`#CC2936`)
- **Functions**: Forest green (`#1B5633`)
- **Strings**: Forest green (`#1B5633`)
- **Constants**: Harvest gold (`#A86B20`)
- **Status Bar**: Red Seal (`#CC2936`)

## Installation

1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X / Cmd+Shift+X)
3. Search for "Perfection Fresh"
4. Click Install
5. Select the theme: File > Preferences > Colour Theme
