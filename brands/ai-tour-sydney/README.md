# AI Tour Sydney Brand

A modern, tech-forward colour palette inspired by the AI Tour Sydney 20-24 April 2026 event.

## Colour Scheme

### Primary Colours
- **Deep Navy**: `#0F2E74` — Main background colour
- **Brilliant Blue**: `#1570EF` — Primary accent and highlights
- **Teal**: `#1E6784` — Secondary accent

### Accent Colours
- **Cyan**: `#C8F9F9` — Keywords, keywords, bright highlights
- **Light Blue**: `#B1DDF5` — Strings, secondary elements
- **Sky Blue**: `#00D9FF` — Terminal cyan

### Neutrals
- **White**: `#FFFFFF` — Primary text
- **Muted Blue**: `#7B95C8` — Comments, secondary text
- **Off-white**: `#E8F0FF` — Body text (dark theme)

## Design Philosophy

The palette draws inspiration from the vibrant, modern aesthetic of the AI Tour event while maintaining professional readiness for code editing. The deep navy provides a sophisticated dark background that reduces eye strain, while the brilliant blues and cyans ensure excellent visibility for syntax highlighting.

## Usage in VS Code Theme

The colour scheme is optimised for:
- Long coding sessions with reduced eye strain
- Clear distinction between code elements
- Professional appearance in any environment
- Terminal usage with full colour support
