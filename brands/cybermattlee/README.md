# CyberMattLee

A neon-on-midnight VS Code colour theme inspired by [CyberMattLee](https://cybermattlee.com) — Matt Lee's hot-magenta beard and deep-purple cyber aesthetic. *Raise the tide of cybersecurity, one keystroke at a time.*

Two variants ship in one extension:

- **Matt Lee Full Beard** — the dark theme. Deep-purple SOC at midnight, lit by the iconic hacker beard.
- **Matt Lee Shaved** — the light theme. Beard off, clean white, magenta accents.

## Features

- Dual variants — **Matt Lee Full Beard** (dark) and **Matt Lee Shaved** (light)
- WCAG AA contrast across syntax tokens
- The magenta beard mark as the Git file icon (and the activity-bar icon)
- Custom file icon theme using the brand palette
- Brand Palette sidebar with one-click HEX / RGB / CMYK copy

## Install

```bash
code --install-extension lucidlabs.cybermattlee-theme
```

Then pick **Matt Lee Full Beard** or **Matt Lee Shaved** from `File > Preferences > Theme > Color Theme`.
