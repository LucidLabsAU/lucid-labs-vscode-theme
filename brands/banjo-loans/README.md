# Banjo Loans

A bold VS Code colour theme featuring electric blue accents and deep purple backgrounds, inspired by the Banjo Loans visual identity.

## Brand Colours

| Colour | Hex | Role |
|--------|-----|------|
| **Electric Blue** | `#0039F6` | Primary brand — buttons, badges, accents |
| **Sky Blue** | `#3D7AF5` | Secondary accent — functions, links (dark) |
| **Deep Purple** | `#30003C` | Dark backgrounds, panels |
| **Burnt Orange** | `#D8613C` | Tags, HTML elements, warm accent |
| **Sage Green** | `#B1C5A4` | Strings (dark), natural accent |
| **Amber** | `#D8A06C` | Constants, warnings |

## Dark Theme

- **Background**: Deep purple-black (`#1A1024`)
- **Text**: Cool off-white (`#E8E4F0`)
- **Keywords**: Bright blue (`#6B9BFF`)
- **Functions**: Electric blue (`#3D7AF5`)
- **Strings**: Sage green (`#B1C5A4`)
- **Constants**: Amber (`#D8A06C`)
- **Errors**: Warm red (`#E06050`)

## Light Theme

- **Background**: Clean white (`#FFFFFF`)
- **Text**: Near-black (`#111111`)
- **Keywords**: Electric blue (`#0039F6`)
- **Functions**: Deep purple (`#30003C`)
- **Strings**: Forest green (`#3D6B30`)
- **Constants**: Burnt sienna (`#A85020`)
- **Status Bar**: Electric blue (`#0039F6`)

## Installation

1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X / Cmd+Shift+X)
3. Search for "Banjo Loans"
4. Click Install
5. Select the theme: File > Preferences > Colour Theme
